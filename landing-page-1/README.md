## Production ready Style and JS are under:

	dist/css & dist/js

## What were considered for this page?

- Semantic HTML
- Accessibility
- W3C Markup Valication

## Prerequisites
- [Node/NPM](https://nodejs.org/en/download/)

## Development

Watch style changes:

	npm run saas

To minify CSS, run:

	npm run build-css

To minify JS, run:

	npm run minify-js

To create a production ready style & JS

	npm run build
