## Prerequisites
- [Node/NPM](https://nodejs.org/en/download/)

## Development

Watch style changes:

	npm run saas

To minify CSS, run:

	npm run build-css

To minify JS, run:

	npm run minify-js

To create a production ready style & JS

	npm run build
